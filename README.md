# 简介
  
  目前正在开发中，目标是符合[MaterialDesign](https://material.io/guidelines/material-design/)风格和MVP模式的第三方Dribbble客户端

## 使用技术（目前）
   
* Android Design库的使用
* 应用MVP模式
* Rxjava的使用
* Glide4的使用
* 内存优化
* Fragment的管理
* 自定义Spinner
